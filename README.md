# АвтоСнаб — Веб-интерфейс для заведений

MVP веб-приложение для оформления заказов заведениями (кафе, рестораны) без работы в Google Таблице.

## Функционал

- Выбор роли: кухня, бар, зал, администратор
- Список избранных товаров с быстрым указанием количества
- Каталог товаров с поиском и добавлением в избранное
- Форма заявки если товар не найден в каталоге
- История заказов со статусами

## Стек

- [Next.js 15](https://nextjs.org) (App Router)
- [Tailwind CSS](https://tailwindcss.com)
- [shadcn/ui](https://ui.shadcn.com)
- TypeScript

## Запуск

```bash
npm install
npm run dev
```

Открой [http://localhost:3000](http://localhost:3000)

## Ветки

- `main` — стабильная версия
- `develop` — рабочая ветка, сюда мерджат фичи
- `feature/*` — ветки для задач
